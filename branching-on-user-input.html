Here's how I route depeneding on user input

  1. set up a generic routing page, here 00-page-routing.html
  2. add an action and method to the form code, here 'action="00-routing" method="get"'
  3. Add id="nextPage" and value="[your-next-page]" to where you want things to change, here input id="post-details" type="radio" name="nextPage" value="02-2-your-overseas-post-details"
  4. Replace the button with an input, here input type="submit" class="button" value="Continue"
  5. This will redirect the user to whatever you set the value to be. Note that if you don't set the value, it will error out
